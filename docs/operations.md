# 運営マニュアル(引き継ぎ資料)

交流会1回分の運営を、このシステムでどう進めるかの手順書です。
**運営者が変わってもこの通りに進めれば同じ運営ができる**ことを目的にしています。

管理画面のURL・Basic認証のID/パスワードは前任者から引き継いでください。

## 全体の流れ

```
⓪日程調整(調整さん・ほぼ自動) → ①イベント作成(取込で自動) → ②アナウンス送信 → (参加ボタンで自動集計)
→ ③日程別グループ作成・紐付け → ④グループ案内送信 → ⑤スライドURL設定・案内送信
→ 前日15:00 自動送信 → 当日9:00 自動送信 → アンケート自動送信 → ⑥完了にする
```

⚙️ 日程調整・イベント作成・前日・当日・アンケートは**ほぼ自動**です。人の作業は②〜⑤のボタン操作が中心で、進捗はすべてイベント詳細ページの**チェックリスト**に表示されます。

## ⓪ 日程調整をする(調整さん)

1. 管理画面 → **日程調整** → **+ 来月の日程調整を開始**
   → 調整さん(chouseisan.com)に来月の全日程の出欠表が自動作成され、URLがメイングループにLINE投稿されます
2. メンバーは調整さんページで ◯/△/× を入力します(アプリ・ログイン不要)
3. 回答が集まったら **結果を取り込んでイベント作成** を押す
   → ◯=1点・△=0.5点で集計した**上位2日程**(同点は早い日付優先)のイベントが自動作成され、イベント詳細に移動します

💡 開始時刻は19:00で作成されます。変更したい場合は日程カードの「開催日時」を編集してください。

## ① イベントを作成する(手動で作る場合)

⓪の取込を使わず手動で作ることもできます:

1. 日程調整の結果を見て、参加人数が多い**2日程**を決める(最終判断は運営者)
2. 管理画面 → イベント一覧 → **+ 新しいイベント**
3. イベント名(例: 7月交流会)と2つの日程・開始時刻を入力して作成

→ どちらの方法でも、作成時点で「前日15:00」「当日9:00」「アンケート(デフォルト当日21:00)」の自動送信が予約されます。アンケート時刻は日程カードの編集フォームで変更できます。

## ② 開催アナウンスを送る

1. イベント詳細のチェックリストで **開催アナウンス** の「送信する」を押す
   (メイングループが未設定だと警告が出ます。初回セットアップはREADME参照)
2. メイングループに日程ごとの**参加ボタン付きメッセージ**が届く
3. メンバーがボタンを押すと、トークには何も表示されずに
   **参加者リストへ自動で記録**される(取り消しボタンもあります)。
   誰が押したかはイベント詳細ページの参加者リストで確認してください

💡 ボタンを押しそびれた人・口頭で参加表明した人は、日程カードの「名前を入力して手動追加」で追加できます。

## ③ 日程別LINEグループを作って紐付ける

1. LINEアプリで日程ごとのグループを作成する(例: 「7/18 交流会」)
2. 作ったグループに**ボットを招待**する → 管理画面の「グループ」ページに自動で現れる
   ※ 2チャネル構成(グループ分散)の場合は、日程別グループには**チャネル2のボット**を招待する(メイングループはチャネル1のボット)。どちらのチャネルが担当かはグループページに表示され、送信時は自動で使い分けられます
3. イベント詳細の日程カードで「日程別LINEグループ」からそのグループを選んで**保存**
4. LINEアプリでグループの**招待リンク**をコピーし、日程カードの「グループ招待リンク」に貼って保存

## ④ グループ案内を送る

チェックリストの **日程別グループ案内** の「送信する」を押す。
メイングループに「7/18(土)参加の方はこちら → 招待リンク」という案内が流れます。

※ 招待リンクから入らない人がいたら、LINEアプリから直接招待してください(ボットにはできない操作です)。

## ⑤ 自己紹介スライドを設定・案内する

1. 日程カードの「自己紹介スライドURL」を入力して保存
2. あわせて「参加方法」(Zoomリンクや会場)と「当日の流れ」も入力しておく
   — **当日案内の自動送信に使われます。未入力だと当日案内が失敗します**(チェックリストに理由が表示されるので、入力して「再送信」すれば大丈夫です)
3. チェックリストの **自己紹介スライド案内** の「送信する」を押す

※ 誰が未記入かの管理はしません。「まだ書いていない人は書いてね」という全体向け文面です。
※ 節約したい月は、この案内を省略しても前日案内にスライドURLが入ります(README「メッセージ数の注意」参照)。

## 自動送信(前日・当日・アンケート)

なにもしなくても以下が送信されます:

| タイミング | 宛先 | 内容 |
|---|---|---|
| 前日15:00 | 日程別グループ | 明日開催・開始時間・スライドURL・未記入者への依頼 |
| 当日9:00 | 日程別グループ | 本日開催・開始時間・参加方法・スライドURL・当日の流れ |
| 当日21:00(変更可) | 日程別グループ | アンケート(1回目用/2回目以降用のURL) |

- 交流会が終わった直後に送りたければ、チェックリストの「今すぐ送る」で前倒しできます
- 送信に失敗すると ❌ と理由が表示されます。設定を直して「再送信」を押してください
- 開催日時を変更すると、未送信の予約時刻は自動で追従します

## ⑥ イベントを完了にする

すべて送信し終わったら「完了にする」を押します(一覧の見た目の整理のためで、押し忘れても実害はありません)。

## トラブルシューティング

| 症状 | 確認すること |
|---|---|
| 日程調整の開始・取込が失敗する | 調整さん(chouseisan.com)の仕様変更の可能性。エラーメッセージに理由が出ます。復旧まで調整さんを手動運用し、①の手動イベント作成に切り替える |
| 日程調整のURLがグループに届かない | 日程調整ページに「⚠️ 未投稿」が出ていれば「グループに投稿する」で再送。メイングループ未設定の場合はグループページで設定 |
| チャネル2のグループに送信できない | Vercelの環境変数 `LINE_CHANNEL_2_ACCESS_TOKEN` / `LINE_CHANNEL_2_SECRET` が設定済みか。チェックリストの❌の理由に変数名が出ます |
| ボタンを押したのに参加者に反映されない | Webhook設定(LINE Developersコンソール)が有効か(チャネル2は `?channel=2` 付きURL)。Vercelのログにエラーがないか |
| 前日/当日案内が時刻になっても届かない | cron-job.org のジョブが動いているか(実行履歴を確認)。チェックリストが ❌ になっていれば理由が書いてある |
| グループがグループ一覧に出てこない | ボットがそのグループに入っているか。公式アカウント設定で「グループ・複数人チャットへの参加を許可」が有効か |
| 送信が「メイングループ未設定」で失敗 | グループページでメイングループを設定したか |
| 管理画面に入れない | Basic認証のID/パスワード(Vercelの環境変数 `ADMIN_USER` / `ADMIN_PASSWORD`) |
